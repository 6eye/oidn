==================

Requirements
------------

Open Image Denoise requires a CPU with at least SSE4.2 support.

The software dependencies are:
* Operating system:
  * Linux
  * Windows 7 or later
* Compiler with C++11 support:
  * GNU Compiler Collection
  * Clang
  * Intel C/C++ Compiler
  * Microsoft Visual C++
* CMake 3.0 or later
* Python 2.7 or later
* Intel Threading Building Blocks (TBB) 2017 or later
